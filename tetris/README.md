## tetris

### Play
| Key | Action |
| - | - |
| ⬅️ ➡️ | Move piece horizontally |
| D ⬆️ | Rotate piece clockwise |
| A | Rotate piece counterclockwise |
| ⬇️ | Speed up piece dropping |
| SPACE | Hard drop piece |
| SHIFT | Switch current piece for piece in storage |
